При удалении файлов и картинок из админ панели django он оставляет их в файловой системе.
Как сделать так что бы django удалял файлы и из каталога в котором они лежат автоматически.

Для этого нужно установить библиотеку django-cleanup.

pip install django-cleanup

в файле settings.py подключаем.

INSTALLED_APPS = (
    ...
    'django_cleanup',
)

При удалении поста буду автоматически удаляться и его файлы и картинки из папки media.

https://github.com/un1t/django-cleanup
